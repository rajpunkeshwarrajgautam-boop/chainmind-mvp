# Vendor due diligence questionnaire (DDQ) — template

Send to subprocessors (hosting, Stripe, observability vendor, IdP). Store completed PDFs + risk rating.

1. **Company & service** — entity name, service description, data touched.  
2. **Certifications** — SOC 2 Type II / ISO 27001 / PCI (if payments). Attach report or bridge letter.  
3. **Subprocessors** — list fourth parties; notification process for changes.  
4. **Security controls** — encryption, key management, logical access, SDLC, vuln scanning frequency.  
5. **Incident notification** — SLA to notify you; sample runbook.  
6. **Data location** — regions; support for data residency commitments.  
7. **Termination / export** — data return & deletion timelines.  

Score each area (Low/Med/High risk) and record compensating controls.
