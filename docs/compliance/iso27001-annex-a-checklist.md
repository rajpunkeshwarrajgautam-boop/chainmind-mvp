# ISO/IEC 27001:2022 Annex A — starter checklist

Use with your ISMS scope statement. Controls A.5.x–A.8.x map loosely to policies below.

- [ ] **A.5 Information security policies** — publish ISMS policy set; version control.
- [ ] **A.8 Asset management** — inventory of repos, cloud accounts, datasets, keys.
- [ ] **A.9 Access control** — RBAC (`viewer`/`planner`/`admin`), API keys hashed, SSO (OIDC routes).
- [ ] **A.12 Operations security** — logging, patching (`supply-chain` workflow), capacity.
- [ ] **A.16 Incident management** — `docs/compliance/incident-response-plan-template.md` executed annually.
- [ ] **A.17 Continuity** — backup/restore drills documented.

Certification requires a registrar audit — templates ≠ certificate.
